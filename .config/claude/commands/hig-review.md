Review the UI for compliance with Apple's Human Interface Guidelines for macOS.

Check the following areas:

## Visual Design
- System fonts and colors usage
- Spacing and layout (8pt grid)
- Dark mode support
- Icon usage (SF Symbols)
- Color contrast

## Controls and Interactions
- Native SwiftUI controls
- Button styles and states
- Keyboard shortcuts
- Menu structure
- Tooltips

## Layout
- Window sizing
- Split view behavior
- Scrollable areas
- Text truncation

## Accessibility
- VoiceOver support
- Keyboard navigation
- Focus indicators
- Dynamic Type
- Color contrast ratios

## Platform Integration
- File handling
- System preferences
- Menu bar integration
- Sandboxing compliance

Provide specific recommendations with:
- File locations
- Current implementation
- HIG-compliant alternative
- Code examples if applicable

Reference the `apple-hig-compliance` skill for detailed guidelines.
